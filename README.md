# ML Computer Vision Projects 🚀

This repository contains a collection of Machine Learning and Computer Vision projects built to solve real-world problems using data-driven approaches.

## 🔍 What’s Included
- Image Classification using Machine Learning
- Feature Engineering (GLCM, statistical features)
- Model Training & Optimization (Random Forest, SVM)
- Handling Large Datasets (10k+ images)
- Dimensionality Reduction (PCA)
- Interactive Web Apps using Streamlit

## 🎯 Goal
To build practical, industry-level machine learning applications with a focus on performance, usability, and real-world deployment.
